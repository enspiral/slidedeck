* Chinese: [audreyt.github.io/whatis.enspiral.com/](https://audreyt.github.io/whatis.enspiral.com/)

## How to

### install

```
git clone https://github.com/enspiral/whatis.enspiral.com
cd whatis.enspiral.com
npm install
```

### start

```
npm start
```

### develop

```
npm run develop
```

### deploy

```
npm run deploy
```
